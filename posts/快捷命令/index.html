ES命令#

  • 查询所有索引

    curl -X GET "http://172.29.1.65:9700/_cat/indices?v"

  • 分词测试

    curl -X GET "http://localhost:9200/globalairport_index/_analyze" -H 'Content-Type: application/json' -d'
{
   "text":"中华人民共和国MN","tokenizer": "ik_max_word"
}'

  • 别名操作

    #查询别名
curl -X GET "http://172.29.1.65:9700/fpr_index_tips202008191131/_alias/*"

#添加别名
curl -X POST "http://172.29.1.65:9700/_aliases" -H "Content-Type: application/json" -d '{"actions": [{"add": {"index": "fpr_index_tips202008191131","alias": "fpr_index_tips"}}]}'

#删除别名
curl -X POST "http://172.29.1.65:9700/_aliases" -H "Content-Type: application/json" -d '{"actions": [{"remove": {"index": "fpr_index_tips202008191131","alias": "fpr_index_tips"}}]}'

  • 查看集群节点信息

    curl -X GET "http://172.29.1.65:9700/_cat/nodes?v=true&pretty"
